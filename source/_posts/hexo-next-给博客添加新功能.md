## <a id="search">添加搜索</a>

要添加搜索功能就得先下 hexo-generator-searchdb

` npm install hexo-generator-searchdb --save `

接着修改 根目录的 _config.yml,在最底部加上如下配置

```js
  #Local Search
  search:
  path: search.xml
  field: post
  format: html
  limit: 10000
```

再同步修改 themes 里的 _config.yml，搜索 local_search 修改enable为true

```js
  local_search:
    enable: true
```
